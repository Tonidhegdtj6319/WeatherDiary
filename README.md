# Training Planner — План тренировок

**Автор:** Неверова Антонина
**Дата** 04.05.2026
**Вариант:** Training Planner

## Описание

GUI-приложение для планирования тренировок. Позволяет:
- Добавлять тренировки с датой, типом и длительностью.
- Фильтровать записи по типу тренировки и по дате.
- Сохранять и загружать данные в JSON.
- Удалять ненужные записи.

## Требования

- Python 3.7+
- Библиотеки: `tkinter` (входит в стандартную поставку Python), `json` (стандартная)
